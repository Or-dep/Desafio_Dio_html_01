strong -- dentro das tags -- Negrito, porem destaca e da ênfase o texto drentro do código


em -- dentro das tags -- Itálico, porem destaca e da ênfase o texto drentro do código


mark -- em determinado texto dentro das tags Texto marcando, para marcar o texto, para alterar a cor coque um style dentro da tag ou crie um style para o mark em head


Para alterar o tamanho do texto use big para textos grandes & small para textos pequenos


Para textos que não devem ser considerados mais use del


Para textos que seram sublinhados usamos a tag ins


Para colocarmos textos sobrescritos usamos o sub ex: 110


Para colocarmos textos subscritos usamos o sup ex: H2O
